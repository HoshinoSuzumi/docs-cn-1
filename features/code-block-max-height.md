# 代码块最大高度

如果代码无法适应一个幻灯片，你可以使用 `maxHeight` 来设置一个固定的高度并启用滚动：

````md
```ts {2|3|7|12}{maxHeight:'100px'}
function add(
  a: Ref<number> | number,
  b: Ref<number> | number
) {
  return computed(() => unref(a) + unref(b))
}
/// ...很多行代码
const c = add(1, 2)
```
````

请注意你可以使用 `{*}` 作为 <LinkInline link="features/line-highlighting" /> 的占位符:

````md
```ts {*}{maxHeight:'100px'}
// ...
```
````
